## 2. Typography & Basics

We use the standard Markdown syntax, refined by our design system.

### Text Hierarchy
*   `##` (H2) for major sections.
*   `###` (H3) for subsections.
*   `**bold**` for highlights.
*   `*italic*` for emphasis.

### Lists

**Ordered List**
1.  First Item
2.  Second Item
3.  Third Item

**Unordered List**
*   Item A
*   Item B
*   Item C

**Nested List**
1.  Parent Item
    *   Child Item
    *   Child Item
2.  Parent Item

## 4. The Layout Engine (Grids & Boxes)

The system uses `CSS Grid` powered by Tailwind.

### The Basic Grid
By default, `<Grid>` creates a **2-column** layout.

<Grid className="my-6">
    <Box className="bg-neutral-100 p-4 rounded">
        **Column 1**

        <p>Content goes here.</p>
    </Box>
    <Box className="bg-neutral-100 p-4 rounded">
        **Column 2**

        <p>Content goes here.</p>
    </Box>
</Grid>

<CodeBlock
    filename="usage-grid-basic"
    code={`<Grid>
    <Box className="bg-neutral-100 p-4 rounded">
        **Column 1**
        Content...
    </Box>
    <Box className="bg-neutral-100 p-4 rounded">
        **Column 2**
        Content...
    </Box>
</Grid>`}
/>

### Controlling Columns
You can override the columns using Tailwind classes: `grid-cols-1`, `md:grid-cols-3`.

<CodeBlock
    filename="usage-grid-cols"
    code={`<Grid className="grid-cols-1 md:grid-cols-3 gap-4">
  <Box>1</Box>
  <Box>2</Box>
  <Box>3</Box>
</Grid>`}
/>

## 5. Advanced Patterns (Bento Grids)

The "Bento" look is achieved by spanning rows and columns.

**Key Requirements:**
1.  Define a fixed row height logic (e.g., `auto-rows-[100px]`).
2.  Use `col-span-X` and `row-span-Y` on the Boxes.

<Grid className="grid-cols-4 gap-2 auto-rows-[100px] my-6">
    <Box className="col-span-2 row-span-2 bg-neutral-800 rounded-lg text-white p-4 font-bold flex items-end">
        2x2 MAIN
    </Box>
    <Box className="col-span-2 row-span-1 bg-neutral-200 rounded-lg p-4 font-mono text-xs">
        2x1 WIDE
    </Box>
    <Box className="col-span-1 row-span-1 bg-neutral-100 rounded-lg flex items-center justify-center">1x1</Box>
    <Box className="col-span-1 row-span-1 bg-neutral-100 rounded-lg flex items-center justify-center">1x1</Box>
</Grid>

<CodeBlock
    filename="usage-bento"
    code={`<Grid className="grid-cols-4 auto-rows-[100px] gap-2">
  {/* Large Square */}
  <Box className="col-span-2 row-span-2 bg-neutral-800 rounded-lg p-4" />
  
  {/* Wide Bar */}
  <Box className="col-span-2 row-span-1 bg-neutral-200 rounded-lg p-4" />
  
  {/* Small Squares */}
  <Box className="col-span-1 row-span-1 bg-neutral-100 rounded-lg" />
  <Box className="col-span-1 row-span-1 bg-neutral-100 rounded-lg" />
</Grid>`}

## 7. Troubleshooting & Gotchas

### Hydration Errors
If you see errors like `Hydration failed` or `Home cannot be a descendant of p`, it means you placed a Block element (like `<div>`, `<Grid>`, or `<Box>`) inside a paragraph.

**The Fix:** Always leave a **blank line** between text and components.

<Grid className="grid-cols-2 gap-4 my-6">
    <Box className="bg-red-50 p-4 rounded border border-red-200">
        **❌ INCORRECT**
        <span className="text-xs font-mono block mt-2 text-red-600">
            Some text
            &lt;div&gt;Block content&lt;/div&gt;
        </span>
    </Box>
    <Box className="bg-green-50 p-4 rounded border border-green-200">
        **✅ CORRECT**
        <span className="text-xs font-mono block mt-2 text-green-600">
            Some text
            <br/><br/>
            &lt;div&gt;Block content&lt;/div&gt;
        </span>
    </Box>
</Grid>
